# 专利与技术文献双向翻译

## 使用指引（Instructions）

**0. 确定翻译方向（前置步骤）**
- 分析用户提供的文本主语言（以字符占比判断）。
- 若文本以英文为主（含中英混杂但英文占主导）：执行 **英译中**。
- 若文本以中文为主（含英中混杂但中文占主导，或明确中文文献）：执行 **中译英**。
- 用户明确指定翻译方向时，以用户指令为准。
- 注意：英译中是最高频场景，本 Skill 默认适配该方向。

**1. 解析原文结构（两种方向通用）**
- 保留原文的所有段落层次、列表、标题等文档结构，不擅自合并、拆分或重排段落。
- 原文中的数字、日期、公式、化学式、代码块、变量名、单位符号等非文本内容，保持原样不动。
- 专有名词（人名、公司名、产品型号等）不翻译，保持原文。

**2. 术语处理**

- **英译中时：**
  - **首次出现的专业术语**：译文后紧跟括号标注原文，格式为 `中文译名（English Term）`。
  - **后续出现的同一术语**：直接使用已确定的中文译名，保持全文术语一致。
  - 对于业界已有通用译名的术语，优先采用通用译名。
- **中译英时：**
  - **首次出现的专业术语**：英文术语后紧跟括号标注中文原文，格式为 `English Term（中文术语）`。
  - **后续出现的同一术语**：直接使用已确定的英文术语。
  - 确保英文术语的学术准确性，优先使用目标领域标准术语表（如 IEEE、IUPAC 等）中的规范表达。

**3. 翻译执行**

- 确保技术含义与原文严格一致，不添加解释性文字，不删减技术细节。
- 禁止使用文学性修饰、比喻等非技术性表达。
- 对翻译后可能产生的歧义或难以简洁表述的概念，以注释形式补充：
  - 英译中：`[译者注：对术语/背景的简要说明]`
  - 中译英：`[Translator’s note: brief explanation]`
- 英文输出需符合学术写作规范，包括正确的冠词、单复数、时态和科技英语的惯用表达（如被动语态、名词化结构）。
除了参考文献 REFERENCES 不用翻译外，但是要标注说明，其它的内容全部需要完整翻译，不要遗漏。
**4. 格式输出**

- 输出格式与输入原文保持一一对应的段落结构。
如果论文有段落或者页面位置也要保留，方便核实对照、查询。
- 表格、列表、标题层级、引用块等格式元素必须原样保留。
- 不额外输出开场白、结束语或翻译说明（除非原文含有需要注释的疑难概念）。
